📘 Hosting Setup Guide

🎯 How to Host Your Admin System

Here are the best options for hosting this tool rental admin system:

Option 1: GitHub Pages (FREE) ⭐ Recommended
  1. Create a free account at github.com
  2. Create a new repository (e.g., "lytton-tool-rentals")
  3. Upload your files: index.html, style.css, app.js
  4. Go to Settings → Pages
  5. Select "main" branch and save
  6. Your site will be live at: yourusername.github.io/lytton-tool-rentals

✅ Perfect for this application, completely free, and easy to update!

Option 2: Netlify (FREE)
  1. Create account at netlify.com
  2. Drag and drop your folder with the 3 files
  3. Site goes live instantly with a free URL
  4. Optional: Connect custom domain for $10-15/year

✅ Easiest option - drag, drop, done!

Option 3: Vercel (FREE)
  1. Sign up at vercel.com
  2. Import project from GitHub or upload files
  3. Auto-deploys on every update
  4. Free SSL certificate included

✅ Great for automatic deployments

Option 4: Budget Hosting (~$5/month)

Namecheap, HostGator, or Bluehost:

  1. Purchase shared hosting plan (~$3-6/month)
  2. Get free domain name (lyttontoolrentals.ca)
  3. Upload files via FTP or cPanel File Manager
  4. Professional custom domain included

💡 Best if you want your own domain name (lyttontoolrentals.ca)

📁 Files You Need to Upload

  • index.html - Main application file
  • style.css - Styling
  • app.js - Application logic

🔒 Important Security Notes

  • ⚠️ This demo stores data in browser memory only
  • ⚠️ For production, change the admin password in the code
  • ⚠️ Consider adding a backend database for permanent storage
  • 💡 All data resets when you refresh the page

🚀 Quick Start (Easiest Path)

  1. Download all three files (index.html, style.css, app.js)
  2. Go to netlify.com
  3. Drag the folder onto Netlify
  4. Done! You'll get a free URL instantly

Need help? All these platforms have excellent documentation and support!
